LibreOffice Writer 添加数学公式、矩阵、特殊字符

Posted on 2015-10-13 Views: 99+

不怎么用Writer，很多功能都不了解，今天写作业要画矩阵。于是上网各种搜，不知道是搜索引擎的问题还是关键词没选好，愣是没找到怎么弄，只有一段代码。无奈之下，一个一个菜单项试，结果第一次就对了。。。这运气。

依次选择菜单栏中的“insert”、“object”，“formula”，这就是输入公式的窗口了

insert object formula

window

矩阵

画一个3*3的矩阵

draw a matrix

font sans bold size *2 A =left[matrix{1#0#0##0#1#0##0#0#1}right]

解释一下：

字符 含义
font sans 用 sans 字体
bold 粗体
size*2 两倍大小字体。标准大小应该是跟上下文一致吧（我认为）。
A 被修饰的部分
= 普通字符
matrix{1#0#0##0#1#0##0#0#1} 3*3 矩阵
# 分隔
## 此行结束
left[、right] 左右中括号。注：经测试，同时存在才有效

再来研究一下其他公式

数学公式 代码 备注
$x_1$ x_1
$x^2$ x^2
$(x_1)^2$ x_1^2
$(x_1)^2+(x_2)^2$ x_1^2+_2^2
$sin^2 x_1$ Sin^2 x_1 注意空格

特殊符号

依次选择“insert”、“special character”，找到想要的符号点击即可，当然，你也可以通过某些输入法来输入

special character

special character

补充

注：修饰默认是相邻的一部分，想要修饰更多可以尝试加大括号“{}”，减少修饰可以尝试一下空格“ ”，或者前面的部分加大括号“{}”。
